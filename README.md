# Reference Easy Editor

仅实现了简单的功能，其中包括
1. 爬取文章作者：输入文章名，即可爬取文章作者，可选缩写作者名
2. 敬请期待


# Usage
## 爬取文章作者
使用方法非常简单，如下：
``` python
from core import reference_do

# 文章标题，或者能在搜索引擎中搜索到的关键词
title = "Attention is all you need"

# 检索文章作者
reference_do.search_authors_by_title(title)
```
